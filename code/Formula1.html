What is the distribution of circuits around the world that have hosted Formula 1 races and how has it changed over time?

Processing of data

The data for this visualization was processed using Python Pandas dataframes. The datasets used were circuits and races from the Formula 1 World Championship (1950 - 2023) dataset on Kaggle. Firstly, null values were checked to ensure the data was ready for processing. Then, the datasets were merged based on their common attribute, the 'circuitId', to associate each circuit with its corresponding season. Next, only the necessary attributes were selected for the visualization, such as the year, circuit name, city, country, latitude, and longitude. A dictionary was then created where each year is a key and the value is a list of dictionaries, each representing a race at a particular circuit for that year. The inner dictionary contains the circuit name, city, country, and the latitude and longitude coordinates. Finally, the resulting dictionary was saved as a JSON file, ready to be used by the D3 JavaScript library.
